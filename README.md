## Task 4: Traffic Accident Data Analysis

### Objective
Analyze the US Accidents dataset to identify patterns related to road conditions, weather conditions, and time of day. Visualize accident hotspots and contributing factors using exploratory data analysis techniques.

### Dataset
- **Dataset:** US Accidents (March 2023)
- **Source:** Kaggle / US Accidents Dataset

### Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

### Project Workflow
- Imported the US Accidents dataset
- Explored the dataset and examined its structure
- Cleaned the data by handling missing values and duplicates
- Performed Exploratory Data Analysis (EDA)
- Analyzed accident trends by:
  - Time of day
  - Day of the week
  - Weather conditions
  - Accident severity
  - State-wise distribution
  - Accident hotspots
- Generated visualizations to identify patterns and contributing factors

### Visualizations
- Accidents by Hour
- Accidents by Day of the Week
- Top Weather Conditions
- Accident Severity Distribution
- Top 10 States with Most Accidents
- Temperature Distribution
- Correlation Heatmap
- Top Accident Hotspots

### Key Insights
- Accident frequency varies throughout the day.
- Weather conditions significantly influence accident occurrence.
- Certain states and cities experience a higher number of accidents.
- Peak traffic hours are associated with increased accident frequency.
- Data visualization helps identify accident hotspots and factors affecting road safety.

### Conclusion
This project demonstrates how Exploratory Data Analysis can be used to uncover meaningful insights from large-scale traffic accident data. The analysis highlights important patterns that can support better traffic management, road safety planning, and accident prevention strategies.
